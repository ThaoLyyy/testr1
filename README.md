###  Getting started:

- Step 1: Clone repository
  + HTTPS:
```bash
git clone https://github.com/ThaoLyyy/react_exercises
```

- Step 2: Change Folder Directory
```bash
cd testr1
```

- Step 3: Checkout branch
```bash
git checkout develop

```

- step 4:
```bash
npm install
```

- step 5 : run project
```bash
npm start
```

- step 6: Open in browser:
```
http://localhost:3000
```

- step 7 : run storybook
```bash
npm run storybook
```
